# Github Issue Viewer

# Install

- 다음 명령어를 터미널창에 순서대로 입력해주세요.
- 명령어 실행 후 http://localhost:3000 에서 어플리케이션을 실행할 수 있습니다.

```jsx
git clone https://github.com/stitchy11/github-issue-viewer.git
cd github-issue-viewer
npm install
npm start
```

.env파일

```jsx
REACT_APP_PERSONAL_TOKEN={"깃허브 access token"}
```
